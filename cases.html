Производство

Цифровая трансформация кадрового учёта

Штат: 250 чел Срок: 3 месяца Год: 2025

Задача: Производственное предприятие вело кадровый учёт «на бумаге». Документы терялись, сроки нарушались, риск штрафов при проверке — критический.

Что сделано: Полный перевод кадрового документооборота из бумажного архива в систему 1С ERP. Внедрена система KPI для HR-процессов. Разработана многоступенчатая программа адаптации.

Результаты

  • Сокращение времени обработки кадровых операций на 60%
  • Повышение удержания новых сотрудников на 35%
  • 100% соответствие трудовому законодательству
  • Исключены риски административных санкций
Сеть / Филиалы

Кадровое управление сетью из 21 филиала

Штат: 250+ чел Филиалов: 21 Период: 2,5 года

Задача: Федеральная сеть фитнес-клубов в разных регионах. Отсутствие централизованного HR-управления. Каждый клуб вёл документы по-своему.

Что сделано: Единоличное управление кадрами всей сети. Разработка и внедрение единых корпоративных стандартов кадрового документооборота для всех 21 объекта.

Результаты

  • Единые стандарты для всех 21 объекта
  • Автоматизация расчётов — 0% ошибок
  • Бесперебойное сопровождение без увеличения HR-штата
  • Полная аттестация рабочих мест с учётом вредных факторов
Охранная деятельность

Построение кадровой службы с нуля для ЧОО с 1000+ сотрудников

Штат: 1000+ чел Срок: 7 лет

Задача: Компания-лидер охранных услуг. Бурный рост, высокая текучесть, сложная лицензионная деятельность. Кадровой службы не было.

Что сделано: Создание кадровой функции с нуля. Все внутренние регламенты. Полный цикл корпоративных изменений: смена гендиректора, перерегистрация акций, изменения устава. Управление ~1000 договорами страхования.

Результаты

  • 0 штрафов за 7 лет при проверках ГИТ, ПФР, ФСС
  • Единственный кадровик на 1000+ человек
  • Полный цикл корпоративных трансформаций «под ключ»
  • Построена воронка найма и резерв кандидатов
Строительство

Кадровое сопровождение при вредных условиях труда

Штат: 200+ чел Вредные: 100 чел

Задача: Группа строительных компаний. 100 человек на вредных условиях труда. Необходим допуск СРО. Работали иностранные высококвалифицированные специалисты.

Что сделано: Полный цикл КДП для группы компаний. Оформление иностранных ВКС. Подготовка документации для СРО. Организация медосмотров, выдача СИЗ, специальная оценка условий труда.

Результаты

  • Получен допуск СРО
  • 0 нарушений при проверках
  • Все иностранные ВКС оформлены без замечаний
  • 100% соблюдение требований по охране труда
